Animation

jQuery has a lot of built-in functions that allow developers to create simple animations with almost no effort. Below are demonstration of simple animations that can be easily implemented using jQuery.

Animation Demo 1 Description: animate() function enables custom animation of a set of CSS properties. Press arrow key to move the image up/down/right/left. Clicking the image will make the image fade out and then fade in again. Whenever mouse leaves an image, the element will be bounced three times.

Animation Demo 2 Description: With animate() and setTimeout() functions, we can easily create an automatic image slider. Every 2 seconds, animate() function moves the image to the left. After we see the last slide, it goes back to the first slide and continues.
