Animation

The jQuery library provides several techniques for adding animation to a web page. These include simple, standard animations that are frequently used, and the ability to craft sophisticated custom effects. We provide several demos that typify how animations work in jQuery

Animation Demo 1 Description: animate() function enables custom animation of a set of CSS properties. Press arrow key to move the image up/down/right/left. We can create many ther effects easily using jQuery such as fadeOut(), fadeIn()..etc. Clicking the image will make the image fade out and then fade in again. Whenever mouse leaves an image, the element will be bounced three times.
