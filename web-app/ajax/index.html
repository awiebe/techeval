Ajax

AJAX allows web applications to interact with the server, without loading a new page. Historically AJAX was one of the core features of jQuery, because browser vendors had not yet decided on the same API. Ajax can be invoked using jQuery in a variety of ways, notably:

$("selector").load("path"); Which replaces the inner html of the target elements with the loaded data.
and
$.get("path",<handler>) Which allows custo handling of response text.
