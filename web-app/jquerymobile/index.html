Designing our music apps, it has become obvious that it's extremely tedious to hide certain elements and display others in an interactive way (when a user clicks a certain element on the page, like selecting a playlist). JQuery mobile is a framework that provides a set of tools to easily create applications like our Music Apps all within a single .html document.

It does so by giving certain elements a data-role element, which defines what kind of element it is. This content is wrapped in a div with a data-role="page" tag and an id specifying what page it will be. This is obviously page 1!

Let's see a different page!

Note that when you click the above button, a 'flipping' animation is performed. If you click the back button in the browser from that other page, you will see it reverses back (which is different than from hitting the 'back' button within the second page, where it instead flashes away).

It's also possible to create modal boxes or even inline popups which allows for fun and interactive possibilities for all web applications.

There's also a large number of widgets you can include into your websites. For example, maybe you want to let users select their current area they're in when they're registering. This can easily be included:

    

Maybe during a login form, you'd provide an option for the server to remember you during a login (keeping your session active).

    

All of these options available in jQuery mobile allow you to make awesome web applications with clean visuals. They are all optimized for mobile device such as smartphones and tablets, but also look great on Desktop browsers.
